# Blockchain-Driven Fraud Detection & Compliance Platform
A technical proof-of-concept demonstrating how distributed ledger technology can replace manual compliance with self-enforcing algorithmic regulation.  

- Cryptographic Data Integrity: Implements a Layer-1 ledger using Keccak256 hashing    to sequentially link blocks and mathematically prevent the injection of synthetic    "Frankenstein" identities.  

- Programmable Compliance: Features an ERC-20 smart contract that evaluates off-       chain risk assessments and automatically executes asset freezes on high-risk         accounts to mitigate threats.  

- Tech Stack: Built and simulated using Python, Web3.py, Solidity, Ganache, and        OpenZeppelin libraries.


The project was built as assigment for a MSc blockchain class so is it not a complete model but only a demo
